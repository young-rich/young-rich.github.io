# Synchronized

## 线程同步

* 在多线程编程时，一些敏感数据不允许被多个线程同时访问，保证数据在任何时刻，最多只有一个线程访问，以保证数据完整性。


## 同步方法-Synchronized

1. 同步代码块
   ```java
    synchronized（对象）{   //得到对象的锁，才能操作同步代码块
        //需要同步的代码块
    }
    ```
2. synchronized还可以放在方法声明中
   ```java
    public synchronized void m（）{
        //同步的代码块
    }
    ```
